Esto es una aplicación de lista de tareas diseñada para ayudar a los usuarios a organizar y gestionar sus actividades diarias.

La aplicación ofrece una serie de funciones que permiten a los usuarios añadir, visualizar, completar, eliminar y editar tareas de manera sencilla y eficiente.

  • Agregar Tarea:

    Permite a los usuarios crear una nueva tarea proporcionando un título y una descripción opcional. Las tareas se añaden a la lista de pendientes.

  • Ver Lista de Tareas:

    Muestra todas las tareas actuales, proporcionando una vista clara de las tareas pendientes y completadas. La lista puede ser filtrada o clasificada según diferentes criterios, como fecha de creación o prioridad.

  • Marcar Tarea como Completada:

    Permite a los usuarios marcar una tarea como completada. Las tareas completadas se destacan de alguna manera para diferenciarlas de las pendientes.

  • Eliminar Tarea:

    Los usuarios pueden eliminar tareas que ya no son necesarias. Esta función es útil para mantener la lista de tareas limpia y relevante.

  • Editar Tarea:

    Permite a los usuarios actualizar la información de una tarea existente, como el título, la descripción o la prioridad, en caso de que haya cambios o errores.
